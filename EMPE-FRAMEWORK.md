# EMPE Framework — Technical Model

EMPE is a four-phase engineering framework.

## Phase 1 — Structure

Generate a Technical Blueprint.

Goal:
Define boundaries and eliminate ambiguity.

Outputs:
- Technical Blueprint
- Scope definition

---

## Phase 2 — Map

Translate UX and data requirements into machine-readable artifacts.

Outputs:
- Mermaid flowchart
- Prisma schema

---

## Phase 3 — Plan

Create an Execution Workspace.

Structure:

docs/
agents/
plans/

AI operates through specialized roles.

---

## Phase 4 — Execute

AI assistants perform implementation guided by deterministic plans.

The developer orchestrates execution.
